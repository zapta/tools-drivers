[Apio](https://github.com/FPGAwars/apio) package that contains the drivers and tool for Windows.

```
package/
├── bin
│   ├── ftdi_eeprom.exe
│   ├── lsftdi.exe
│   ├── lsusb.exe
│   ├── serial_install.exe
│   └── zadig.exe
├── package.json
└── share
    └── zadig.ini
```

The director 'package' is compressed as the 'drivers' package file.
